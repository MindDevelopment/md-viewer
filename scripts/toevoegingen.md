# Toevoegingen / Features

- **Live preview** — renderen gebeurt automatisch tijdens typen (zonder op knop te drukken), met een korte vertraging (debounce)
- **Dark/Light theme toggle** — schakelen tussen donker en licht thema
- **Exporteer als HTML** — de geproduceerde HTML downloaden of kopiëren
- **Auto-save** — tekst blijft bewaard in localStorage, ook als je de pagina per ongeluk sluit
- **Table of Contents** — automatische inhoudsopgave op basis van headings
- **Emoji ondersteuning** — `:smile:` wordt 😄
- **Themakiezer** — preview styling wisselen (GitHub, GitLab, Nordic, etc.)
- **Printen** — de preview netjes afdrukken
